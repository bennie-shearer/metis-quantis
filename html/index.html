Metis Quantis

Welcome to the Metis Quantis, a web-based interface for financial calculations powered by QuantLib. This application provides a user-friendly way to perform various quantitative finance calculations, including bond pricing, option valuation, swap analysis, and yield curve fitting.

Key Features

Bond Analytics

Calculate prices, yields, duration, convexity, and cashflows for fixed-rate and zero-coupon bonds.

Option Pricing

Value options using Black-Scholes, binomial trees, Monte Carlo, and finite difference methods.

Swap Valuation

Analyze interest rate swaps, calculate NPV, fair rate, and cashflows for both legs.

Yield Curve Building

Construct yield curves using bootstrapping from market instruments or parametric models.

Curve Fitting

Fit yield curves to bond data using various methods like Nelson-Siegel and Svensson.

Interactive Visualization

Visualize results with interactive charts for better understanding and analysis.

Yield Curve Fitting Methods

This toolkit provides several methods for fitting yield curves to bond data:

  • Nelson-Siegel - A parametric model with three factors (level, slope, curvature)
  • Exponential Splines - Fitting with exponential basis functions
  • B-Splines - Local polynomial spline-based fitting
  • Svensson - An extension of Nelson-Siegel with an additional hump parameter
  • Svensson (Filtered) - Svensson model with smoothing techniques

To get started:

  1. Select one of the tools from the list above
  2. Enter the required parameters for your calculation
  3. Click the Calculate button to see the results
  4. Explore various visualizations and data tables for in-depth analysis
